# 知识库概述

## 核心主题
该知识库主要覆盖中国境内的企业财税、工商、社保及人力资源等领域的实务操作与政策咨询，服务于企业日常经营中的合规与税务筹划场景。

## 关键知识点
- **税务管理**：涵盖增值税、企业所得税、个人所得税的计算、申报、发票开具（普票/专票/红冲）及税收优惠政策（如小规模纳税人免税、高新技术企业优惠）。
- **社保与公积金**：涉及社保开户、缴纳、补缴、封存、退工流程，以及公积金操作和残保金政策。
- **工商注册与变更**：包括公司注册、经营范围变更、法人/股东变更、地址异常处理及注销流程。
- **发票与票据处理**：解答发票开具规则（税率、差额开票）、票据报销合规性（如个人抬头发票）、以及进项销项匹配问题。
- **纳税信用与风险**：说明纳税信用等级（如A级、D级）的影响、税务稽查重点以及税务风险防范。
- **薪酬与个税**：涵盖工资申报、个税计算（起征点、税率跳档）、年终奖申报以及退休返聘人员薪酬处理。
- **特殊资质与备案**：涉及外经证、进出口备案、医疗器械备案、危化品运营资质等专项许可。
- **实缴资本与减资**：解释注册资本实缴新规、实缴期限及减资操作流程。
- **公户与私户资金往来**：规范公转私、报销、付款的合规要求与风险提示。

## 适用问题类型
- 企业日常经营中遇到的税务计算、申报及优惠政策咨询。
- 社保、公积金的开户、缴纳、变更及补缴等操作流程问题。
- 发票开具、收取、红冲以及各类票据报销的合规性判断。
- 工商注册、信息变更、地址异常处理及公司注销相关咨询。
- 薪酬发放、个人所得税计算及申报的具体操作问题。
- 企业各类经营资质（如进出口、外经证）的申请与使用咨询。

## 不适用场景
- 涉及国际税法（非中国境内）的深度筹划与复杂跨境税务问题。
- 具体的司法诉讼、劳动仲裁案件代理或详细的法律条文解读。
- 个人投资理财、股票证券、金融市场分析等非企业运营类金融问题。
- 行业前沿技术研发、产品设计、市场营销策略等非财税工商核心业务问题。
