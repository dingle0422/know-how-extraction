# 知识库概述

## 核心主题
该知识库主要覆盖中国境内的企业工商、税务、社保、公积金等综合性财税与合规咨询服务领域，服务于中小企业在日常经营中遇到的各类实务操作与政策咨询场景。

## 关键知识点
1. **税务申报与发票管理**：涵盖增值税、企业所得税、个人所得税的申报规则，以及发票开具、红冲、作废、抵扣等具体操作。
2. **社保与公积金实务**：包括社保开户、增员减员、补缴、转移、退休办理，以及公积金开户、扣款、基数调整等全流程操作。
3. **企业设立与变更**：涉及公司注册、经营范围变更、法人/股东变更、注册资本实缴与减资、地址异常处理等工商事务。
4. **税收优惠政策与风险**：解析小规模纳税人免税政策、高新技术企业优惠、留抵退税等，并提示税务稽查重点与信用等级（如A/D级）影响。
5. **财务合规与账务处理**：指导公户与私户支付规范、成本票取得、费用报销标准、个税计算与申报、以及常见账务问题处理。
6. **特殊资质与行业监管**：咨询医疗器械备案、危化品运营资质、建筑劳务资质、进出口备案、外经证开具等特定行业许可要求。
7. **用工与薪酬福利**：解答劳动合同、年假、加班费、离职补偿、工伤、生育津贴、残疾人就业安置等劳动法规相关问题。
8. **企业注销与异常处理**：指导公司注销流程，以及应对工商异常、税务非正常户、法律仲裁等风险事件的解决方案。

## 适用问题类型
1. **操作流程类**：例如“如何办理社保退工？”、“公司变更法人需要什么材料？”、“发票开错了怎么办？”。
2. **政策解读类**：例如“小规模纳税人季度30万免税政策包含专票吗？”、“高新技术企业有哪些增值税优惠？”、“残保金如何减免？”。
3. **合规判断类**：例如“个人抬头的发票能否报销？”、“公户转私户在什么情况下合规？”、“长期0申报有什么风险？”。
4. **税务计算类**：例如“开XXX元发票需要交多少税？”、“工资发多少不用交个税？”、“卖旧车涉及哪些税费？”。
5. **风险咨询类**：例如“纳税信用等级D级有什么后果？”、“被税局抽查到该怎么办？”、“地址异常对经营有什么影响？”。

## 不适用场景
1. **个人非经营性税务问题**：不涉及纯粹的个人所得税汇算清缴（如个人稿酬、偶然所得）、个人房产交易税等与公司经营无关的个税细节。
2. **具体司法诉讼与复杂法律文书**：不提供具体的法律诉讼代理服务、合同起草或深度法律意见，仅涉及常见的劳动仲裁、工商举报等风险提示。
3. **境外（除简要提及外）及特定尖端行业财税**：不深入覆盖复杂的国际税收筹划、跨境并购，或金融、证券等受特殊监管行业的专业财税政策。
