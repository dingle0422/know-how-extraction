# 业务主题一：合同生效的法律要件审查

## 1. 业务定性与前置判断
- **触发条件**：处理任何合同签订、审核或争议处理事务时，均需启动本审查流程。
- **核心目标**：确保合同具备完全的法律效力与可执行性，防范无效、可撤销或无法强制执行的风险。

## 2. 标准处理 SOP / 解答思路
1.  **第一步：形式要件审查**
    - **签署主体**：确认签约双方具备相应的民事权利能力和行为能力。
    - **意思表示**：确认签署行为是双方真实意思的体现，无欺诈、胁迫等情形。
    - **签署方式**：合同必须通过法定或约定的方式确认，通常包括：
        - 双方签字。
        - 加盖单位公章或合同专用章。
        - 法律认可的其他方式（如可靠的电子签名）。
2.  **第二步：实质要件审查**
    - **内容合法性**：逐条审视合同条款，确保其不违反法律、行政法规的强制性规定。
    - **公序良俗**：合同目的及内容不得违背社会公共利益和善良风俗。
    - **权责对等**：审查核心权利义务条款（如价款、交付、违约责任）是否明确、公平。

## 3. 特殊场景与例外分支
- **格式条款**：提供格式条款的一方应遵循公平原则，并采取合理方式提示对方注意免除或限制其责任的条款，否则该条款可能无效。
- **批准/登记生效**：法律、行政法规规定需办理批准、登记等手续才生效的合同，自手续完成时生效。

## 4. 政策依据
《中华人民共和国民法典》第一百四十三条、第四百六十五条、第四百九十条、第四百九十六条等。

# 业务主题二：增值税进项税额认证与抵扣操作

## 1. 业务定性与前置判断
- **适用主体**：增值税一般纳税人。
- **核心关切**：如何将取得的合规增值税专用发票用于当期进项税额抵扣，以准确计算应纳税额。
- **前置条件**：取得的发票必须为合规的增值税专用发票（或符合规定的其他抵扣凭证），且相关购进业务用于增值税应税项目。

## 2. 标准处理 SOP / 解答思路
1.  **第一步：发票取得与初步审核**
    - 从销售方取得注明增值税税额的专用发票（包括发票联和抵扣联）。
    - 初步审核发票信息（购买方信息、货物服务名称、金额、税额等）是否准确无误。
2.  **第二步：平台勾选确认（认证）**
    - **操作平台**：登录国家税务总局指定的官方电子平台（如**增值税发票综合服务平台**）。
    - **核心操作**：在平台中对计划用于当期抵扣的发票进行“勾选”或“签名确认”操作。此步骤即完成发票的认证/确认，是向税务机关申报抵扣信息的前置程序。
3.  **第三步：申报抵扣**
    - **时限要求**：必须在**规定的纳税申报期内**（通常为次月1日至15日，遇节假日顺延）完成勾选确认及纳税申报。
    - **申报操作**：在办理增值税纳税申报时，准确填写《增值税及附加税费申报表附列资料（二）》（本期进项税额明细），系统将自动带入已勾选确认的发票数据，计算当期可抵扣进项税额。

## 3. 特殊场景与例外分支
- **发票丢失**：丢失增值税专用发票抵扣联的，可凭销售方提供的相应专用发票记账联复印件，作为增值税进项税额的抵扣凭证或退税凭证。
- **逾期认证/确认**：增值税一般纳税人取得的增值税专用发票，未在规定期限内（原为360天，现无具体天数限制，但需在纳税申报期内勾选）进行认证或确认的，不得作为合法的增值税扣税凭证，不得计算进项税额抵扣。
- **不得抵扣情形**：用于简易计税方法计税项目、免征增值税项目、集体福利或者个人消费的购进货物、劳务、服务、无形资产和不动产，其进项税额不得从销项税额中抵扣。

## 4. 政策依据
《中华人民共和国增值税暂行条例》及其实施细则、国家税务总局关于增值税发票综合服务平台等事项的公告（如国家税务总局公告2020年第1号）及相关征管规定。
